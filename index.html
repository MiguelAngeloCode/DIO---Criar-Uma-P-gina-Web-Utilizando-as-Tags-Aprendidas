Introdução ao HTML

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas da Web.
É uma linguagem fácil de aprender e usar, mesmo para iniciantes. Com HTML,
você pode criar estruturas de páginas da Web, adicionar conteúdo e estilizar elementos da página.

Estrutura Básica do HTML

Um documento HTML básico consiste em duas tags principais: <html> e <body>.
A tag <html> envolve todo o conteúdo da página da Web, enquanto a tag contém o conteúdo principal visível para os usuários.

Elementos HTML

Os elementos HTML são blocos de construção que compõem o conteúdo de uma página da Web.
Existem diferentes tipos de elementos HTML, cada um com sua própria função específica.
Alguns elementos comuns incluem:

  • Cabeçalhos (<h1>, <h2>, ..., <h6>): Criam títulos e subtítulos
  • Parágrafos (<p>): Contêm blocos de texto
  • Listas (<ul>, <ol>): Criam listas ordenadas ou não ordenadas
  • Links (<a>): Criam links para outras páginas da Web ou recursos
  • Imagens (<img>): Inserem imagens em uma página da Web

Atributos HTML

Os atributos HTML são usados para fornecer informações adicionais sobre os elementos.
Eles são especificados dentro das tags de abertura do elemento, usando o formato atributo="valor".
Por exemplo, o atributo src da tag <img> especifica o caminho para o arquivo de imagem.
